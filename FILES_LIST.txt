📋 قائمة الملفات في مجلد GitHub
═══════════════════════════════════════════════════════════════

تأكد من رفع جميع هذه الملفات لـ GitHub:

📁 github/
│
├── 📁 .github/                         ⚠️ مجلد مهم جداً!
│   └── 📁 workflows/
│       └── monitor.yml                 ✅ الـ workflow
│
├── 📄 crash_monitor_github.py          ✅ السكريبت الرئيسي
├── 📄 .gitignore                       ✅ تجاهل ملفات غير مهمة
├── 📄 README.md                        ✅ صفحة الـ repo
├── 📄 GITHUB_SETUP.md                  ✅ دليل كامل مفصّل
├── 📄 START_HERE.txt                   ✅ دليل سريع (ابدأ هنا!)
├── 📄 HOW_TO_UPLOAD.txt                ✅ دليل الرفع
└── 📄 FILES_LIST.txt                   ✅ هذا الملف

═══════════════════════════════════════════════════════════════

⚠️ ملاحظات مهمة:
═══════════════════════════════════════════════════════════════

1. المجلد .github يبدأ بنقطة (.)
   - في Windows: قد يكون مخفياً
   - لإظهاره: View → Hidden items ✅

2. تأكد من رفع الـ workflow:
   .github/workflows/monitor.yml
   بدون هذا الملف، Actions لن تعمل!

3. يمكنك رفع جميع الملفات دفعة واحدة:
   - حدد كل الملفات والمجلدات
   - اسحبهم لصفحة GitHub

═══════════════════════════════════════════════════════════════

✅ بعد الرفع، ستجد في الـ repo:
═══════════════════════════════════════════════════════════════

📄 README.md           - صفحة رئيسية جميلة
📄 crash_monitor.py    - الكود
📁 .github/            - مجلد Actions
📄 GITHUB_SETUP.md     - الدليل الكامل
📄 START_HERE.txt      - دليل سريع

وبعد تشغيل أول workflow:
📄 crash_records.json  - البيانات JSON
📄 crash_records.csv   - البيانات CSV (للـ Excel)
